# Code of Conduct

This project follows a simple standard: be respectful, technical, and constructive.

## Expected Behavior

- Discuss ideas and code on their merits.
- Provide clear reproduction steps for bugs.
- Assume good intent, but be direct about technical risks.
- Keep security-sensitive reports private until resolved.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Publishing private keys, secrets, or exploit details from another user's report.
- Spam or off-topic promotion.

## Enforcement

Maintainers may edit, hide, or remove comments, issues, and pull requests that violate this code of conduct. Repeated or severe violations may result in a ban from participating in the project.
